# Ortografía simplificada del español.

## Descripción
Se deben seguir las normas ortográficas tradicionales del español, salvo en los casos específicos definidos a continuación. Las reglas de acentuación (tildes) y los signos de puntuación se mantienen sin cambios.

** Esta norma no aplica para unidades de medida, nombres propios o extranjerismos.**

 ### Vocales

 Todos los sonidos vocálicos se representan con las letras *a, e, i, o* y *u*. De este modo, la letra “y” o “i griega” pasa a usarse únicamente como consonante. Por ejemplo, “soy” pasa a escribirse “soi”.

 ### B y V

 Todos los sonidos representados por /b/ se deben escribir con *b*. Así, “vida” pasa a ser “bida” y “vaca” pasa a ser “baca”.

 ### C, K, Q

 Todos los sonidos /k/ se representan con la letra *k*. De este modo, “casa” pasaría a ser “kasa” y “que” a ser “ke”.

 ** Dado que la *c* es el carácter dominante para este fonema, se esperaría que todos los sonidos /k/ se representaran con esta letra. Sin embargo, como la *c* también representa los fonemas /se/, /si/ o /θe/, /θi/, resultaría contraintuitivo usarla para representar /ke/ y /ki*. Además, podría convertirse en un obstáculo a la hora de leer textos con ortografía tradicional.

 ### H
 
 La *h* solo debe usarse en el dígrafo “ch”; en cualquier otro caso, debe omitirse. De este modo, la palabra “hilo” pasa a ser “ilo” y “hacha” a ser “acha”.
 
 ### G y J
 
 Para el sonido /x/ siempre se debe usar la *j*. De este modo, “corregir” pasa a ser “correjir” y “gente” a ser “jente”.
 
 ### LL y Y
 
 Todos los sonidos /ʝ/ (como en “ya”) o /ʒ/ (como en el “sho” argentino) deben escribirse con *y*. Así, “llave” pasa a ser “yabe” y “lluvia” a ser “yubia”.
 
 ### C, S y Z
 
 Todos los sonidos /s/ deben escribirse con *s*.  
 Todos los sonidos /θ/ deben escribirse con *z*.
 
 En España y en aquellos lugares donde se usa el fonema /θ/, “zapato” seguiría escribiéndose igual y “crecer” pasaría a “krezer”. En Latinoamérica, al no existir dicho fonema, “zapato” pasaría a ser “sapato” y “crecer” también pasaría a “kreser”.

 ### X y CC
 Todos los sonidos /ks/ y /kθ/ deben escribirse con X. De este modo, “examen” se mantiene igual, “texto” también, y palabras como “acción” pasan a escribirse “axión”.
 

## Adopción

Esta propuesta puede adoptarse de forma progresiva y flexible en distintos contextos: desde el uso cotidiano hasta entornos formales y técnicos. A continuación se sugieren algunas formas de implementación.

###  Uso cotidiano

- Puedes aplicar la ortografía simplificada en mensajes informales, notas personales o redes sociales.
- No es necesario anunciar su uso, aunque puedes hacerlo si deseas mayor claridad.

#### Redes sociales.
```text
ose: oi boy a ir a la kasa mas tarde
```
---

###  Proyectos de software

En entornos técnicos, es recomendable **declarar explícitamente** el uso de esta ortografía para evitar confusiones.

#### 🔹 En comentarios de código

```js
/*
* @ortografia: OSE v1
* Esta función korrije la desbiasión estandar del texto
*/
function corregirTexto(texto) {
// implementación...
}
```

#### En documentos formales.

Se puede indicar el uso de esta ortografía mediante un encabezado o un pie de página. Sin embargo, se recomienda el uso de un encabezado para evitar confusiones desde el inicio del documento.

```text
Este email/documento/proyecto utiliza la Ortografía Simplificada del Español (OSE v1):
https://github.com/hatimsue/ose/
```
